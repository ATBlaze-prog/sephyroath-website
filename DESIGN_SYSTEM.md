# SephyrOath Design System Theme

This theme file documents the design system for the SephyrOath Gaming Platform.

## Color Palette

### Primary Colors
- **SO Dark** (#0A0E27): Main background color
- **SO Darker** (#050812): Darker background for depth
- **SO Primary** (#FF6B35): Primary accent (orange-red)
- **SO Primary Light** (#FF8C00): Lighter orange accent
- **SO Neon Red** (#FF5722): Neon red for highlights

### Secondary Colors
- **SO Gold** (#D4AF37): Premium gold text
- **SO Gold Light** (#E8C547): Lighter gold
- **SO Purple** (#9D4EDD): Secondary accent

### Neutral Colors
- **SO Gray 100** (#F0F0F0): Lightest gray
- **SO Gray 200** (#E0E0E0): Light gray
- **SO Gray 300** (#C0C0C0): Medium-light gray
- **SO Gray 400** (#808080): Medium gray
- **SO Gray 500** (#606060): Dark gray

## Typography

- **Headings**: Poppins font family
- **Body**: Inter font family

## Effects

- **Glow Effects**: Red/gold glowing shadows for premium feel
- **Glassmorphism**: Backdrop blur effects for layering
- **Animations**: Smooth transitions and framer-motion effects
- **Particle Canvas**: Background particle effects

## Components

All components follow the SephyrOath dark theme with neon accents and maintain premium esports aesthetic.
